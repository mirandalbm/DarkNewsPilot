---

Título do Fluxo no SIM.AI: DarkNews-SoloMode

Objetivo: Criar, dublar, publicar e promover completamente automaticamente 10 vídeos diários de notícias no estilo "dark" em múltiplos idiomas e canais, exigindo menos de 30 minutos de supervisão diária do usuário.

Filosofia: Automação máxima, controle estratégico mínimo via dashboard intuitivo e relatórios de performance.

---

Fluxo Definitivo de Automação

Acionador: Diário, às 06:00 UTC.

Módulo 1: Pesquisa e Triagem Inteligente (AI-1)

· Ferramenta: Google News API ou NewsData.io + GPT-4-Turbo.
· Instrução para a IA: "Atue como editor-chefe. Colete as 20 notícias mais importantes do dia. Classifique-as com uma nota de 0 a 100 com base no potencial viral, impacto e relevância para um canal de tom 'dark' e misterioso. Priorize palavras-chave como 'vazamento', 'segredo', 'revelado', 'alerta'. Retorne um JSON com as top 10, incluindo título, fonte, URL, resumo objetivo e a nota."
· Saída: top_news.json (com notas de prioridade).

Módulo 2: Criação de Roteiro com Tom "Dark" (AI-2)

· Ferramenta: Claude 3 Opus ou GPT-4-Turbo.
· Instrução para a IA (por notícia): "Escreva um roteiro de vídeo de 45-60s no estilo 'Documentário Mystery'. Estrutura: 1) Introdução impactante com uma pergunta retórica (15s); 2) Exposição factual e direta dos eventos (20s); 3) Conclusão que instiga reflexão ou deixa um questionamento no ar (10s). Inclua instruções básicas de edição para a IA de vídeo, como '[CENA: Stock footage de arquivos secretos]'. Mantenha o tom sério, factual e levemente conspiratório."
· Saída: script_{id_da_noticia}.txt.

Módulo 3: Narração Profissional (AI-3)

· Ferramenta: API ElevenLabs (Voz: Male Dark Narrative - Adam).
· Configurações: Estabilidade: 40%, Clareza +70%, Estilo de Exagero: 30%.
· Ação: Gera um arquivo de áudio MP3 a partir do roteiro.
· Saída: narration_{id_da_noticia}.mp3.

Módulo 4: Renderização do Avatar (AI-4) - (SEM EDIÇÃO MANUAL)

· Ferramenta: API Synthesia ou HeyGen.
· Ação: Utiliza um template de cenário pré-configurado (ex: Cenario_Escuro_1). Envia o arquivo de áudio e o texto do roteiro para a API, que retorna um vídeo MP4 renderizado com o avatar sincronizado.
· Saída: video_main_{id_da_noticia}.mp4 (Vídeo principal pronto).

Módulo 5: Sistema de Multi-Dublagem Automatizada (AI-5)

· Ferramenta: API ElevenLabs Dubbing Studio + SIM.AI Video Tool.
· Idiomas Pré-configurados: Português-BR, Espanhol-ES, Espanhol-MX, Alemão, Francês, Hindi, Japonês, Inglês-EUA.
· Ação: Para cada video_main_{id}.mp4:
  1. A API do ElevenLabs traduz e gera a narração em cada idioma, mantendo a mesma voz.
  2. O SIM.AI Video Tool substitui a faixa de áudio original e adiciona uma legenda com a bandeira do país.
· Saída: video_{id}_{lingua}.mp4 (ex: video_01_br.mp4).

Módulo 6: Dashboard de Aprovação e Retrabalho (Portal Humano)

· Ferramenta: Dashboard nativo do SIM.AI.
· Funcionalidades:
  · Lista de vídeos com miniatura, título e dropdown de idioma.
  · Botões APPROVE e EDIT.
  · Botão EDIT abre um menu com opções de retrabalho:
    · [ ] Regenerate Script
    · [ ] Regenerate Voiceover
    · [ ] Change Avatar/Template
    · [ ] Custom Instruction
  · Aprovação move o vídeo para Approved_For_Publishing. Edições acionam o retrabalho automático apenas da etapa selecionada.

Módulo 7: Publicação Multi-Canal Automatizada (AI-6)

· Ferramenta: YouTube API v3 + GPT-4 (para SEO).
· Ação (por vídeo aprovado):
  1. IA gera título, descrição e tags otimizadas no idioma correto.
  2. O sistema publica automaticamente no canal do YouTube correspondente ao idioma (ex: vídeo BR -> canal Brasil).
· Configuração: Os canais são pré-mapeados no SIM.AI (Canal Principal, Canal Brasil, etc.).

Módulo 8: Criação e Publicação de Shorts (AI-7)

· Ferramenta: RunwayML ou HeyGen Shorts + API do YouTube.
· Ação: Identifica o clipe de 15-30s de maior impacto de cada vídeo, reformata para vertical, adiciona legendas dinâmicas e publica como Short no canal correspondente.

Módulo 9: Distribuição Viral e Cross-Promotion (AI-8)

· Ferramenta: Hootsuite/Buffer API + GPT-4.
· Ação: Gera e publica posts adaptados para Twitter, Facebook, Instagram e Reddit com o link do vídeo, impulsionando o tráfego.

Módulo 10: Motor de Crescimento e Análise (AI-9, AI-10, AI-11)

· AI-9 (SEO): TubeBuddy/vidIQ API para otimizar metadados antes da publicação com base em tendências.
· AI-10 (Engajamento): Responde automaticamente a comentários simples no YouTube via API, seguindo regras conservadoras para evitar erros.
· AI-11 (Análise): Consome YouTube Analytics API e gera um relatório semanal simplificado para o usuário, destacando:
  · Top 5 vídeos por performance.
  · Idioma com melhor retenção.
  · Sugestões de conteúdo (ex: "Notícias de tecnologia têm 50% mais views").
  · Este relatório realimenta a AI-1 para melhorar a seleção de notícias.

Instrução Final para o SIM.AI Copilot:

"SIM.AI Copilot, crie um fluxo de trabalho robusto chamado DarkNews-SoloMode baseado na estrutura detalhada acima.

Princípios Fundamentais:

1. Autossuficiência: Todo o processo de criação de vídeo deve ser feito via APIs de renderização (Synthesia/HeyGen), eliminando a necessidade de qualquer edição manual de vídeo.
2. Controle Eficiente: O dashboard de aprovação deve ser o centro de controle, permitindo aprovação com um clique ou retrabalho direcionado via menu de opções, mantendo o fluxo intacto.
3. Gestão Multi-Canal Automática: A publicação deve ser direcionada automaticamente para o canal de YouTube correto com base no idioma do vídeo, gerenciando vários canais sem intervenção.
4. Melhoria Contínua: O sistema deve aprender com as métricas de performance. O relatório da AI-11 deve ser usado para ajustar automaticamente os critérios de pesquisa da AI-1, criando um ciclo de otimização.

O resultado final deve ser um sistema que me permita supervisionar a operação em menos de 30 minutos por dia, focando apenas em aprovação e ajustes estratégicos, enquanto a IA executa toda a operação repetitiva de criação, publicação e promoção."
